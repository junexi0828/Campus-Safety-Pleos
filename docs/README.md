## 📝 문서 업데이트

문서를 업데이트할 때는 다음을 확인하세요:

1. **일관성**: 다른 문서와 일관된 형식과 용어 사용
2. **정확성**: 코드 예제와 실제 구현이 일치하는지 확인
3. **완전성**: 필요한 모든 정보가 포함되어 있는지 확인
4. **가독성**: 명확하고 이해하기 쉬운 설명
